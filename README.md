# コツァ'sブログアプリ

1.概要<br>
このアプリは記事を投稿、閲覧することができるブログアプリです。<br>
特に技術系の記事に向いていますがいろんなジャンルの記事を投稿することができます。<br>

2.開発の背景・経緯<br>
このアプリを開発しようと考えたきっかけは自分が普段記事を読んだりwebサービスを理由するなかで感じた不満を解決しようと考えたからです。<br>
詳しくは後の機能紹介や工夫点で書きますが、目安箱や記事の要約など私が普段から欲しいと考えていた機能を今回のアプリでは実装しました。<br>

3.公開URL<br>
<a href="https://next-blog-app-lake.vercel.app">https://next-blog-app-lake.vercel.app</a><br>

## 特徴と機能の説明

1.各ページについての説明<br>
- 閲覧用記事一覧ページ<br>
このページでは投稿記事が一覧で確認できます。<br>

- 記事詳細ページ<br>
このページでは記事の本文及び内容を楽しむことができます。<br>

- 管理投稿記事一覧ページ<br>
このページでは投稿記事の一覧に加えて、削除や編集ページへの遷移などもできます。<br>

- 投稿記事新規作成ページ<br>
このページでは新たに記事を作成することができます。<br>

- 記事編集ページ<br>
このページでは既存記事の編集をすることができます。<br>

- 管理カテゴリページ<br>
このページではカテゴリを一覧で確認することが可能で、直接削除や編集ページへの遷移が可能です。<br>

- カテゴリ新規作成ページ<br>
このページでは新たにカテゴリを作成することが可能です。<br>

- カテゴリ編集ページ<br>
このページでは既存のカテゴリの編集をすることができます。<br>

- 管理機能一覧ページ<br>
このページでは記事やカテゴリの新規作成、管理用一覧ページ、目安箱のページなど主要な管理用ページにURL入力不要で遷移することができます。<br>

- ヘッダー<br>
ヘッダーはどのページを閲覧していても利用したいページだと考えられる
サイトの説明ページやログインページ、トップページ(閲覧用記事一覧ページ)に遷移することができる。また、目安箱はこのサイトに対する感想や意見などをタイトルと本文に分けて書き、送信することができる。<br>
 感想、意見にも禁則ワードを指定することによって不適切な発言を防いだ。<br>

- サイト説明ページ<br>
このページはこのサイトや製作者である私について説明するページです。<br>
私の自己紹介やこのサイトの簡単な説明、私のXやgithubなどのリンクを記載しています。<br>

- 目安箱ページ<br>
目安箱ページでは送信された意見や感想を管理することができます。<br>

- ログインページ<br>
ログインページではメールアドレスとパスワードを用いてログイン、またはメールアドレスとパスワードを用いずにゲストログインを行うことができます<br>
ゲストログインの場合は編集や新規作成などの一部機能が使用できなくなります。<br>
<br>
2.機能、特徴についての説明<br>

- 閲覧用や管理用記事一覧ページの名前、カテゴリ検索機能<br>
この機能は記事をカテゴリ、もしくは名前で検索することができます。
この検索機能は完全一致ではなく「投稿」と検索したら「投稿1」、「投稿2」が検索できるなど一部の文字が含まれていたら検索できるようにすることで検索のしやすさを向上させました。<br>
この機能によって特定の記事を探したい時に速やかに記事を発見することができます。また、特定の記事ではなく○○についての記事を探している時などはカテゴリ検索を用いることで速やかに記事を発見できます。<br>

- 閲覧用や管理用記事一覧ページとカテゴリ一覧、編集、新規作成ページのソート機能<br>
この機能は記事やカテゴリを新しい順、古い順、タイトル順にソートすることができます。
新しい順、古い順はその名の通り作成された時期によって並び替えます。タイトル順は
カタカナ、数字、英語、ひらがなの順番で並び替えます。<br>
この機能により探している記事を見つけやすかったり、新たな記事を発見することができます。<br>

- 閲覧用の記事一覧ページのいいね、閲覧数ランキング<br>
いいねや閲覧数などでにランキングを実装しました。
1位は王冠マーク、2,3位は銀色、銅色にすることによって視覚的に順位をわかりやすくしました。また、1位,1位,3位のように同じ順位も実装しました。
ランキングに実際の数値を表示することによってランキングのスケールが分かるようにしました。
さらにランキングを見て検索しなくてもいいようにランキングの記事の名前から直接記事の詳細画面に飛べるようにしました。<br>
この機能によって満足度の高い記事、注目されている記事を見つけることができます。<br>

- 閲覧用記事一覧ページの記事の要約<br>
記事をすべて読まなくても記事の内容が分かるように要約機能を実装しました。
この要約は記事を作成するときもしくは編集時に作者が設定することができます。
 要約は150文字以内とすることによっては長くなりすぎないようにしました。<br>
この機能によって1目で大まかに記事を理解することができ、要約を読んで興味を持った結果本文を読みに行くなど記事への導入も行うことができます。<br>



- 閲覧用記事一覧ページに各記事のいいね数とコメント数を表示<br>
各記事のいいねやコメント数を表示しました。<br>
この機能によっていいね数やコメント数を表示することでその記事の注目度などを知ることができ、読む記事を決めやすくすることができます。 また、いいねやコメント数を表示することで記事の作者の達成感などにもつながります。<br>

- 閲覧用や管理用記事一覧ページに各記事のカテゴリを表示<br>
各記事のカテゴリを表示しました。<br>
この機能によって内容や要約を読まずともなんとなくどのような記事かをユーザーは知ることができます。 これによってユーザーの興味を引くことができます。<br>

- 閲覧用や管理用記事、カテゴリ一覧ページのページネーション機能<br>
カテゴリや記事が8個以上になるとページを増やすようにしました。
8の倍数ずつページは増えていき下のボタンでページを切り替えることができます。<br>
この機能によって記事やカテゴリなどの数が増えてきたときに多くスクロールする必要がなくなります。 また、一ページことに表示する項目を減らすことによってページ自体の見やすさを向上させています。<br>

- 記事詳細ページの記事のいいねやコメントを追加する機能<br>
いいねは一度クリックするといいねを付与、もう一度押すことでいいねを解除することができます。いいね付与状態ではハートに色が付け視覚的にわかりやすくしています。
コメントは入力欄に入力、送信ボタンをクリックすることで記事にコメントを投稿することができます。 コメントにも同じ方法でいいねを行うことができます。 <br>
いいねによって作者に記事に対する評価を示すことができます。
コメントによって作者はユーザーの反応を知ることができ、感想やフィードバックを得ることができます。 コメントのいいねにより参考になったり役に立ったコメント、評価の高いコメントを知ることができます。<br>
コメントには禁則文字が設定されていて不適切な発言などはそもそも送信できないようにしました。 これによってコメントによって不快な気持ちになってしまうことを防ぎました。<br>

- 記事の詳細ページに成果物URLを表示<br>
なにか記事の内容に付随する成果物などがある場合は作者が記事の作成または編集時に設定することで記事詳細ページの上部に成果物のURLを表示することができます。
リポジトリやアプリURLなど自由に設定できます。<br> この機能によって自分の成果物を直接ユーザーに示すことができます。また、文中に文章としてではなくクリックすることでページ遷移ができるリンクボタンとしてURLを示すことができます。<br>

- 記事詳細ページに画像を表示<br>
記事詳細ページの上部に画像を表示することができます。<br>
この機能によってその記事がどんな記事なのかを視覚的に知ることができます。 また、今から読む記事がどんな記事なのかを視覚的に知ることで記事の内容がイメージしやすくなります。<br>

- 記事詳細ページにおすすめ記事を表示<br>
その詳細ページの記事と同じカテゴリが含まれている記事を5個表示します。
また、同じカテゴリが含まれている記事が5個未満の場合は投稿されている記事からランダムに補充し、同じカテゴリの数が多ければ多い記事ほど優先して選出されます。<br>
この機能によってユーザーは興味のある記事を連鎖的に読むことができます。<br>

- 管理用記事一覧ページに記事新規作成、管理機能一覧リンクボタンを実装<br>
記事新規作成、管理機能一覧リンクボタンそれぞれのボタンをクリックすることでそれぞれのページに飛ぶことができます。<br>
この機能によって記事の新規作成ページと管理機能がまとめられているページに直接飛ぶことができます。 この二つは記事一覧ページにいるときに行きたいページだと考えたので実装しました。<br>

- 管理用記事一覧、カテゴリ一覧ページに編集、削除ボタンを実装した。<br>
削除ボタンはその記事やカテゴリを編集画面に入ることなく削除することができます。また、間違いで押してしまったときなどに対応するために本当に消すかの確認を実装しました。
編集ボタンはクリックすることでその記事やカテゴリの編集画面に飛ぶことができます。<br>

- 管理用記事一覧、カテゴリ一覧ページの表示方法切り替え機能<br>
縦1列、縦2列のボタンを押すことで記事一覧、カテゴリ一覧の表示方法を切り替えることができます。<br>
固定ではなくユーザーが選択できるようにすることでユーザーの好みに合わせて表示方法をカスタマイズすることができます。<br>



- 管理用記事新規作成ページと記事編集ページに管理用記事一覧、管理機能一覧リンクボタンを実装<br>
記事編集、管理機能一覧リンクボタンそれぞれのボタンをクリックすることでそれぞれのページに飛ぶことができます。<br>
この機能によって記事の編集ページと管理機能がまとめられているページに直接飛ぶことができます。 この二つは事新規作成や編集ページにいるときに行きたいページだと考えたので実装しました。<br>


- 管理用記事新規作成ページと記事編集ページの成果物URL、要約に対して形式などを案内する機能<br>
要約や成果物URLを入力するときに成果物URLは「http」もしくは「https」の形式である必要があることを表示し、要約は文字数を右下に表示します。<br>
この機能によって要約はあと何文字書くことが可能かを知ることができ、構成がしやすくなると考えられます。成果物URLは入力した後に間違いに気づくみたいなことを防ぐことができます。
また、この２つの項目は独自のものなので入力欄にどのようなことを書けばいいかを示しました。<br>

- 管理用記事新規作成ページと記事編集ページに画像を表示する機能<br>
記事詳細ページに表示する画像を選択した時、プレビューとして画像を小さく表示します。<br>
この機能によって画像を選択した時の間違いなどに素早く気づくことができるようになります。<br>

- 管理用記事新規作成ページと記事編集ページのカテゴリ表示方法を変更する機能<br>
この機能はカテゴリの上にある2列、3列ボタンによって記事編集、新規作成ページのカテゴリ表示方法を切り替えることができます。<br>
この機能によって2列か3列ユーザーは好きな表示方法を選択することができます。また、カテゴリ数の増加にも3列で対応することができます。<br>

- 管理用記事新規作成ページと記事編集ページのカテゴリを名前検索する機能<br>
カテゴリ検索欄によってカテゴリの名前でカテゴリを検索することができます。
この検索機能は完全一致ではなく「カテゴリ」と検索したら「カテゴリ1」、「カテゴリ2」が検索できるなど一部の文字が含まれていたら検索できるようにすることで検索のしやすさを向上させました。<br>
この機能によってカテゴリ数が増えてきたり、特定のカテゴリを探す時に全体で探すよりも素早く目的のカテゴリを見つけることができます。<br>


- 管理用記事編集ページのコメント管理機能<br>
記事編集ページからその記事についたコメントを削除できるようにしました。<br>
この機能によって禁則文字などで弾けなかった不適切なコメントを削除することができるようにしました。<br>

- 管理用記事編集ページとカテゴリ編集ページの削除機能<br>
記事、カテゴリ編集ページ下部にある削除ボタンからその記事もしくはカテゴリの削除を行うことができる。
一覧ページからも削除できますが、詳細を見て削除するかどうかを決める場合もあると考えられたので編集ページにも削除ボタンを追加しました。<br>

- 管理用カテゴリ一覧ページ、カテゴリ新規作成ページと編集ページのカテゴリ名前検索機能<br>
カテゴリ検索欄によってカテゴリの名前でカテゴリを検索することができます。
この検索機能は完全一致ではなく「カテゴリ」と検索したら「カテゴリ1」、「カテゴリ2」が検索できるなど一部の文字が含まれていたら検索できるようにすることで検索のしやすさを向上させました。<br>
この機能によってカテゴリ数が増えてきたり、特定のカテゴリを探す時に全体で探すよりも素早く目的のカテゴリを見つけることができます。<br>

- カテゴリ新規作成ページと編集ページの名前被り防止機能<br>
既存のカテゴリと同じ名前を設定しようとしたときそれを知らせ、防止する機能を実装しました。<br>
この機能によって同じ名前のカテゴリが複数作成されるのを防ぐことができます。<br>

- カテゴリ新規作成ページと編集ページの既存のカテゴリ一覧表示機能<br>
カテゴリの新規作成、編集ページに既存カテゴリを一覧で表示する機能を実装しました。<br>
この機能によって入力する前から同じ名前のカテゴリを作成することを防ぐことができます。
また、ここに表示されているカテゴリをクリックすることでそのカテゴリの編集画面に飛ぶこともできます。<br>

- 目安箱ページの送信されたものを対応済みかどうかで管理する機能<br>
送信された意見、感想は対応済みか否かで管理されていて、対応済みもしくは未対応と書かれているところをクリックで状態を変更することができます。 
また、意見、感想は上部の対応済みの一括削除ボタンによって対応済みとなっているものを一括で削除できます。<br>

- 目安箱ページの表示切り替え機能<br>
上部のボタンによって表示する意見、感想を絞り込むまたは、全てを表示することができる。
意見、感想の状態によって表示を切り替えます。<br>

- ログインページのメールアドレス形式誘導機能<br>
メールアドレスを入力するときにメールアドレスの形式に沿っていなかった場合知らせます。この機能によってメールアドレスの入力ミスを軽減することができます。<br>

- ログイン機能<br>
ログインすることで記事やカテゴリの編集や削除、作成などログイン限定機能の実行が可能となります。<br>
